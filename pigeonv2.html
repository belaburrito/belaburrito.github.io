Pigeon v2

Pigeon is a collectible card game developed by Juggler's Mistake, an art collective based in London.


Posters of all 52 Pigeon cards will be posted around the streets, accompanied by a QR code. Scanning the QR code allows you to digitally "collect" the card. If a user scans all the QR codes, they are mailed a physical pack of Pigeons cards.


WIP. Pigeon v2 is a revamp of a project I started in 2022, now intended for deployment in production. Other improvements include:

  • Mobile first, responsive design
  • Deployment with Vercel
  • Backend powered by Supabase, with Postgresql database and database functions
  • Admin tooling for maintenance and improved experience

Future improvements:


Pigeon can be aesthetically improved using the NES.css framework. I am also interested in creating more robust testing and monitoring for the Supabase project.
